# Data Gathering

This file tracks where the project's data came from.

If a source is not currently known, it should stay listed as `to-be-determined` until we confirm it.

## Datasets

### `data/european-countries.json`

- Base dataset source: `to-be-determined`
- Corruption field source: Transparency International, Corruption Perceptions Index 2025
- Link: https://www.transparency.org/en/cpi/2025
- Notes: The `corruption` value in the game data is stored on an inverted scale where higher means more corrupt (`100 - CPI score`). A few microstates without a published CPI entry were filled with manual proxy values during implementation.
